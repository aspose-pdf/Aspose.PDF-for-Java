# Aspose.PDF for Java PHP Plugin

This folder is reserved for PHP integration examples and plugin-specific assets.

## Layout

Directory | Description
--------- | -----------
`src/main/php` | PHP source files.
`sample-data` | Input and output files used only by the PHP plugin examples.
`tools` | PHP plugin helper scripts.

## Build

This module currently uses a placeholder Maven `pom` packaging so it can participate in the repository-level build. Add plugin-specific build steps here when PHP examples are implemented.
