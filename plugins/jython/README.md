# Aspose.PDF for Java Jython Plugin

This folder is reserved for Jython integration examples and plugin-specific assets.

## Layout

Directory | Description
--------- | -----------
`src/main/jython` | Jython source files.
`sample-data` | Input and output files used only by the Jython plugin examples.
`tools` | Jython plugin helper scripts.

## Build

This module currently uses a placeholder Maven `pom` packaging so it can participate in the repository-level build. Add plugin-specific build steps here when Jython examples are implemented.
